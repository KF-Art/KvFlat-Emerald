# KvFlat-Emerald
Kvantum dark theme based on KvFlatTeal. With darker menus.

<H1>Installation</H1>
  
  #Clone this repo
  git clone https://github.com/KF-Art/KvFlat-Emerald/
  
  #Install theme.
  cd KvFlat-Emerald
  cp Solid ~/.config/Kvantum
  cp Blur ~/.config/Kvantum
  
Alternatively, you can install it using Kvantum GUI.
